Descarga

Linux:

  1. Descargamos en http://exelearning.net/descargas la versión adecuada a nuestro sistema operativo, en este caso Linux.
  2. Guardamos el archivo en el disco duro.
  3. Instalamos el paquete descargado: haciendo doble clic sobre el archivo .deb se lanzará el gestor de instalación (gdebi, centro de software Ubuntu...). o  ubicando la terminal en la carpeta donde se descargo el archivo y usar el  comando:  
    sudo dpkg -i intef-exe_2.5_all.deb
    siendo este caso el nombre del archivo intef-exe_2.5_all.deb que corresponde a la version 2.5 de exelearning , en caso de ser una version distinta solo cambie el nombre intef-exe_2.5_all.deb por el del archivo descargado, o en el caso de usar snap usando el comando: 
    sudo snap install exelearning
  4. Al finalizar el proceso se creará un acceso directo en la sección "Educación".

  5. Ejecutamos eXeLearning por primera vez. Nos aparecerá la ventana de preferencias en donde podremos seleccionar el idioma y elegir el navegador sobre el que queremos trabajar. En el caso de no querer mostrar la ventana cuando se incia eXe, desmarcaremos la casilla.

    Recuerda Guardar los cambios.

    Con estos cinco sencillos pasos, nuestro eXeLearning estará instalado y configurado en el idioma deseado.

Windows:

  1. Descargamos en http://exelearning.net/descargas la versión adecuada a nuestro sistema operativo, en este caso Windows.
  2. Guardamos el archivo en el disco duro.
  3. Instalamos el paquete descargado: haciendo doble clic sobre el archivo .exe se lanzará nuestro gestor de instalación.
  4. Al finalizar el proceso se creará un acceso directo dentro del menú "Inicio".

  5. Ejecutamos eXeLearning por primera vez. Nos aparecerá la ventana de preferencias en donde podremos seleccionar el idioma y elegir el navegador sobre el que queremos trabajar. En el caso de no querer mostrar la ventana cuando se incia eXe, desmarcaremos la casilla.

    Recuerda Guardar los cambios.

    Con estos cinco sencillos pasos, nuestro eXeLearning estará instalado y configurado en el idioma deseado.

Macintosh

  1. Conectarse a la pagina web http://exelearning.net/descargas y descargarse el instalable, en nuestro caso para Apple.
  2. Guardar el instalable en nuestro disco duro.
  3. Pinchar en el archivo .dmg, y en la ventana que se abre, arrastrar el icono de eXe a la carpeta "Aplicaciones".
  4. Al finalizar el proceso, eXeLearning estará en la carpeta "Aplicaciones" y estará listo para ser usuado.

  5. Ejecutamos eXeLearning por primera vez. Nos aparecerá la ventana de preferencias en donde podremos seleccionar el idioma y elegir el navegador sobre el que queremos trabajar. En el caso de no querer mostrar la ventana cuando se incia eXe, desmarcaremos la casilla.

    Recuerda Guardar los cambios.

    Con estos cinco sencillos pasos, nuestro nuevo eXeLearning estará instalado y configurado en el idioma deseado.
